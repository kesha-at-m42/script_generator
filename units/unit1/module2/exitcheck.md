## **1.8 EXIT CHECK (3-4 minutes)**

**Detail Level:** Full interaction specification (AI formats and tags)

### **Purpose**

Verify Lesson understanding before Practice. Tests whether student can understand scale meaning, determine correct symbol counts (including half-symbols for odd numbers), create scaled picture graphs, and read values from completed graphs.

### **Parameters**

| Element | Specification |
| :---- | :---- |
| Problems | 4 |
| Cognitive Types | IDENTIFY and CREATE (per Playbook: Modules 1-3 focus) |
| Time | 3-4 minutes |
| Tone | Calm, low-stakes, neutral |
| Remediation | Full L-M-H added via Remediation Pipeline |

### **Constraints**

| MUST | MUST NOT |
| :---- | :---- |
| Use scaled picture graphs (1:2 scale) | Use 1:1 scale or other scales |
| Use Numeric Table | Introduce new data display formats |
| Include both even AND odd values | Use only even values |
| Require half-symbols | Skip half-symbol assessment |
| Use same interaction types as Lesson | Add new interaction types |
| Match Lesson difficulty (values 6-14) | Use values outside Lesson range |

### **Alignment Check**

| Problem | Cognitive Type | Tests | Lesson Source |
| :---- | :---- | :---- | :---- |
| EC.1 | IDENTIFY | Scale meaning ("scale of 2" \= each symbol shows 2\) | Section 1: Interaction 1.2 |
| EC.2 | IDENTIFY | Calculate symbols needed for odd number | Section 2: Interactions 2.1-2.2 |
| EC.3 | CREATE | Place symbols including half-symbol | Section 2: Interactions 2.3-2.4 |
| EC.4 | IDENTIFY | Read total value from scaled graph | Section 1: Interaction 1.3; Section 2: 2.5 |

---

### **Transition into Exit Check**

- **Visual:** Lesson graph clears. Clean transition.
- **Guide:** "You made scaled picture graphs with whole symbols and half-symbols. Let's see what you know."

---

**Problem EC.1: Scale Meaning (IDENTIFY)**

- **Visual: Picture Graphs (Mode 2: Creating).** Vertical. Context: "Favorite Sports," 4 categories. Key: "Each ⭐ \= 2 votes." Values: Soccer=8 (4 symbols), Basketball=6 (3 symbols), Baseball=10 (5 symbols), Tennis=4 (2 symbols). Pre-made (not student-created).
- **Visual: Data Tables (Numeric Display).** Same values displayed alongside.
- **Guide:** "Look at the key. This graph has a scale of 2\. What does 'scale of 2' mean?"
- **Prompt:** "What does 'scale of 2' mean?"
- **Student Action:** \[Multiple choice:
  - "There are 2 categories"
  - "Each symbol shows 2 votes"
  - "Add 2 to find the total"
  - "There are 2 symbols"\]
- **Correct Answer:** "Each symbol shows 2 votes"
- **On Correct:** "Scale of 2 means each symbol shows 2 votes."
- **On Incorrect:** \[Remediation Pipeline\]

**Design Note:** Data table is included so students see that table values (8, 6, 10, 4\) differ from symbol counts (4, 3, 5, 2). This reinforces why the key matters.

**Alignment:** Tests same concept as Lesson 1.2 (key/scale meaning).

---

**Problem EC.2: Symbol Calculation for Odd Number (IDENTIFY)**

- **Visual: Data Tables (Numeric Display).** Shows odd value for one category. Key context: scale of 2\.
- **Visual: Picture Graphs (Mode 2: Creating).** \[Verify: is this a creation task or MC?\] Data Table visible.
- **Guide:** "Emma collected 9 stickers. How many symbols does Emma need?"
- **Prompt:** "How many symbols does Emma need?"
- **Student Action:** \[Multiple choice: 4, 4½, 5, 9\]
- **Correct Answer:** 4½
- **On Correct:** "4 and a half symbols. 9 is odd—4 full symbols for 8, plus a half-symbol for 1."
- **On Incorrect:** \[Remediation Pipeline\]

**Alignment:** Tests same skill as Lesson 2.1-2.2 (determining symbols for odd numbers).

---

**Problem EC.3: Create with Half-Symbol (CREATE)**

- **Visual: Data Tables (Numeric Display).** Data for creation task. Mix of even and odd values.
- **Visual: Picture Graphs (Mode 2: Creating).** Empty or partially filled graph template. Key: "Each \[symbol\] \= 2." Student creates.
- **Guide:** "Place the symbols for Ava. She collected 7 stickers."
- **Prompt:** "Place symbols for Ava."
- **Student Action:** Hover and click to place 3 full symbols \+ 1 half-symbol
- **Correct Answer:** 3½ symbols placed
- **On Correct:** "3 and a half symbols for 7\. Each full symbol is 2, the half is 1."
- **On Incorrect:** \[Remediation Pipeline\]

**Alignment:** Tests same skill as Lesson 2.3-2.4 (creating with half-symbols).

---

**Problem EC.4 : Read Total Value (IDENTIFY)**

- **Visual: Picture Graphs (Mode 2: Creating).** Completed graph (from EC.3 or new pre-made). Key visible. Data Table not visible.
- **Guide:** "Look at the completed graph. How many students take the bus to school?"
- **Prompt:** "How many students take the bus to school?"
- **Student Action:** \[Multiple Choice: 3, 4, 6, 8\]
- **Correct Answer:** 8
- **Answer Rationale:**
  - 8 \= Correct (4 symbols × 2 \= 8\)
  - 4 \= Error: Counted symbols only (Misconception \#1)
  - 3 \= Counting or misread error
  - 6 \= Counting or misread error
- **On Correct:** "8 students take the bus to school. 4 symbols, each worth 2."
- **On Incorrect:** \[Remediation Pipeline\]

**Alignment:** Tests same skill as Lesson 1.3 and 2.5 (reading values from scaled graphs).

---

### **Exit Check Closure**

- **Visual:** Completed graph remains briefly.
- **Guide:** "You understand scaled picture graphs. Let's connect what you learned."

### **Verification Checklist**

**Structure:**

- [ ] Exactly 4 problems
- [ ] Transition references Lesson content with low-stakes framing
- [ ] Closure statement present (brief, no overpraise)
- [ ] Total time 3-4 minutes

**Alignment:**

- [ ] EC.1 tests scale/key meaning (Lesson 1.2)
- [ ] EC.2 tests symbol calculation for odd numbers (Lesson 2.1-2.2)

**Constraints:**

- [ ] Uses scaled picture graphs (1:2 scale only)
- [ ] Uses numeric data table (matches Lesson Section 3\)
- [ ] Includes both even and odd values
- [ ] Requires half-symbol creation (EC.3)
- [ ] IDENTIFY and CREATE only (no COMPARE per Modules 1-3 guidance)
- [ ] Difficulty matches Lesson (values in 6-14 range)
- [ ] No new visual models or formats introduced
