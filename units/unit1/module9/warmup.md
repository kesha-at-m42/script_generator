## 1.6 WARMUP (\~3-4 minutes)
**Interaction Design Reminder:** Every interaction specifies both Guide dialogue and a Prompt. Guide provides teaching context + complete instruction. Prompt contains instruction only. Both must independently allow student to complete the task.
### Core Purpose
Activate expression knowledge from M8 and prime the skip-counting-as-accumulation mechanism that Section 1 of the Lesson will name, extend, and formalize. Students experience the animated accumulation (Context Visualization + synced Equation Builder) without the conceptual bridge being stated — they SEE skip-counting building products, but the Lesson is where they'll HEAR "that was multiplication."
*Key Function:* This Warmup positions M9 as the natural next step from M8. Students arrive knowing how to BUILD expressions (M8 Exit skill). The Warmup activates that skill (W.1), then demonstrates what happens when you extend equal groups with accumulation (W.2) — priming the mechanism the Lesson will name. W.2 also generates the first 5 entries of the Products Strip display, which the Lesson extends to 10.
### Parameters
| Element | Specification |
| --- | --- |
| Interactions | 2 meaningful interactions + bridge |
| Active Participation | 1 of 2 requires student response (W.1); W.2 is observation only |
| Format | Expression Callback (M8 skill) → Animated Accumulation (observation) → Bridge |
| Cognitive Load | 20-30% (W.1 is prior skill; W.2 is observation only) |
| Remediation | Pipeline (light) |
| Tone | Medium-High energy, curious, building toward "keep going" |
| Engagement Anchors | Quick Success (W.1 — familiar expression building), Visual Intrigue (W.2 — animated accumulation creates curiosity) |
---
### Constraints
| USE | AVOID |
| --- | --- |
| Informal language ("Let's warm up," "Interesting, right?") | Formal phrasing ("Observe the following accumulation") |
| Observable acknowledgments ("5 groups of 2 gave us these products") | Assumed states ("You see the pattern") |
| M8 expression format for W.1 (no = sign, no product slot) | Equation format in W.1 (= sign introduced visually in W.2 but NOT named) |
| Bags context for W.1 (primes Lesson Section 2 worked example) | New contexts not yet used in M7-M8 |
| Circles with dots for W.2 (semi-abstract, primes Lesson Section 1) | Real-world contexts for accumulation (save for Lesson) |
| Guide narrates what's happening on screen ("1 group of 2, that's 2") | Strategy explanation ("This is called skip-counting") |
| Products Strip appears as display (numbers visible) | Guide names any pattern ("all even," "multiples") |
| = sign visible in W.2 Equation Builder display | Drawing attention to = sign ("See the equals sign?") |
| "Groups of" as informal description | × symbol name, "multiplication," "factor," "product," "equation," "multiples" |

### Interaction W.1: Expression Callback \[ACTIVATION\]
- **Purpose:** Activate M8 expression building skill before introducing product computation in Lesson.
- **Visual: Context Visualization (Bags with Objects).** 4 bags with 2 items each. Equation Builder appears alongside with expression template: `[___] × [___]` (two slots, no = sign — M8 format).
- **Guide:** "Let's warm up. Remember building these? 4 bags with 2 in each. Build the expression."
- **Prompt:** "Build the expression for 4 groups of 2."
- **Student Action:** Equation Builder Method A (MC) or Methods C/D (tiles). Tile palette: 2, 4, 6, 8.
	- **Options:** \[A: 4 × 2 ✓\] \[B: 2 × 4\] \[C: 4 + 2\] \[D: 2 + 2 + 2 + 2\]
- **Correct Answer:** 4 × 2
- **On Correct:** "4 times 2. You know how to show groups with expressions."
- **Remediation:** Pipeline
> **Design Note:** Pure M8 callback. Low cognitive load — student has done this before. Uses M8's expression format (no = sign, no product slot). Activates "I know how to write expressions" before the Lesson adds "now let's find what they EQUAL." Bags context primes Section 2 worked example.
---
### Interaction W.2: Animated Accumulation — 5 Circles \[OBSERVATION + PRIMING\]
- **Purpose:** Prime the accumulation mechanism through visual animation before Lesson names skip-counting as multiplication.
- **Visual (Phase 1 — Animated Accumulation):** Context Visualization clears. Circles with 2 dots each appear one at a time (\~1.5 seconds per circle), system-driven. Equation Builder updates in sync:
| Circle | Equation Builder | Guide |
| --- | --- | --- |
| 1 circle (2 dots) | `1 × 2 = 2` | "1 group of 2. That's 2." |
| 2 circles (2 dots each) | `2 × 2 = 4` | "2 groups of 2. That's 4." |
| 3 circles | `3 × 2 = 6` | "6..." |
| 4 circles | `4 × 2 = 8` | "8..." |
| 5 circles | `5 × 2 = 10` | "10." |
- **Visual (Phase 2):** Circles remain. A Products Strip appears below:
```plain text
[ 2 ] [ 4 ] [ 6 ] [ 8 ] [ 10 ]
```
- **Guide:** "5 groups of 2 gave us these products: 2, 4, 6, 8, 10."
- **Student Action:** None (observation — system-driven animation)
> **Design Note:** Full narration on circles 1-2 establishes the mechanism (groups accumulating, equation updating, skip-count building). Circles 3-5 let the visual carry — Guide just voices the running total. This prevents W.2 from feeling like instruction while clearly demonstrating the accumulation process. Products Strip appears but Guide does NOT name any pattern (no "all even," no "multiples"). The Equation Builder shows the = sign and product here, which is new relative to W.1's expression-only format — but no attention is drawn to it. Students see it; the Lesson will name it.
---
### Bridge to Lesson
- - **Guide:** "Interesting, right? You've been building expressions — now you're going to find out what they EQUAL. And there are some cool patterns hiding in the answers. Let's keep going."
- **Visual:** Products Strip \[2\] \[4\] \[6\] \[8\] \[10\] and 5 circles remain on screen through transition. (If technical constraints require a screen reset, Lesson 1.1 re-displays them.)
- **Student Action:** None
> **Design Note:** Bridge creates anticipation ("keep going") without teaching. The "interesting, right?" is a curiosity hook, not a question requiring response. Warmup ends with the accumulation paused at 5 — Lesson picks up exactly where it stopped.

### Verification Checklist
**Structure:**
- [ ] Hook in first 15-20 seconds (W.1 — "Let's warm up. Remember building these?")
- [ ] 2 meaningful interactions (W.1 expression callback, W.2 animated accumulation)
- [ ] 1+ judgment/noticing task requiring thought (W.1 — build expression from visual)
- [ ] Bridge creates anticipation without teaching ("Interesting, right? Let's keep going with that.")
- [ ] Total time 3-4 minutes
**Engagement Anchors:**
- [ ] Quick Success (W.1 — familiar M8 expression building, high confidence)
- [ ] Visual Intrigue (W.2 — animated accumulation creates curiosity about what's happening)
- [ ] Personalization (W.1 — "Remember building these?" callbacks to M8 experience)
**Constraints:**
- [ ] M8 expression format for W.1 (no = sign, no product slot)
- [ ] NO "multiples," "equation," "factor," or "product" (Lesson vocabulary)
- [ ] NO pattern formalization (Guide does NOT say "all even" or name patterns)
- [ ] NO strategy explanation ("This is called skip-counting")
- [ ] NO attention drawn to = sign in W.2 (visible but unaddressed)
- [ ] Bags context for W.1, circles for W.2 (both from M7-M8)
- [ ] "Groups of" used informally (not formal vocabulary)
- [ ] Session-relative language only ("Remember building these?" not "Yesterday you...")
- [ ] Light remediation only (W.1 only — W.2 has no student action)
- [ ] All tasks use prior skills (W.1 = M8 expression building; W.2 = observation)
- [ ] Cognitive load 20-30%
**Lesson Alignment:**
- [ ] W.1 expression callback (4 × 2) activates Equation Builder familiarity — students ready for expanded template in Lesson 2.1
- [ ] W.1 bags context primes Lesson 2.1 worked example (5 bags of 2)
- [ ] W.2 accumulation mechanism (5 circles) sets up Lesson 1.1 extension to 10 circles
- [ ] W.2 Products Strip \[2, 4, 6, 8, 10\] persists into Lesson 1.1 for seamless continuation
- [ ] W.2 skip-counting narration primes Lesson 1.3 bridge ("That WAS multiplication")
- [ ] W.1 values (4 × 2) differ from W.2 values (1-5 × 2) — no exact repetition at same cognitive level
- [ ] Bridge defers all vocabulary, pattern naming, and = sign teaching to Lesson (proper scaffolding sequence)
