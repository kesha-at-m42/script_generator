## 1.6 WARMUP
**Interactions:** 2 meaningful interactions + bridge
**Active Participation:** 2 of 2 require student response
**Format:** Judgment task (compare visuals) → Recognition task (match expression) → Bridge
**Cognitive Load:** 20-30% (light — uses M7 skills only)
**Remediation:** Light only
**Tone:** Medium energy, curious, building anticipation
**Engagement Anchors:** Narrative Setup (mystery/curiosity), Choice/Agency (judgment task), Personalization (callback to M7)
### Warmup Constraints
| USE | AVOID |
| --- | --- |
| Session-relative language ("last time/this time") | Temporal language ("yesterday/today") |
| "Groups" / "groups of" (M7 vocabulary) | "Expression" as formal term (Lesson 1.1 vocabulary) |
| "Times" and × symbol (M7 carryover) | "Build the expression" (Lesson teaches this) |
| Informal acknowledgments ("You matched it") | Overpraising ("Perfect!", "Amazing!") |
| Recognition via Method A (M7 familiar tool) | Methods C/D tiles (Lesson introduces these) |
| Values 2-5 groups, 2-6 items (within M8 range) | Complex values or products \>30 |
| Bags and boxes contexts (concrete, from M7) | Circles (semi-abstract, saved for Lesson transfer) |
| "Matching" language (M7 skill) | "Building" language (M8 new skill, bridge only) |
---
**Opening Hook (15-20 seconds)**
- **Visual:** TWO Context Visualizations appear side-by-side:
	- Left: 3 bags with 5 items each
	- Right: 5 bags with 3 items each
- **Guide:** "Check this out. Two different pictures—but something interesting is hiding here."
- **No student action** (curiosity/activation only)
---
### Interaction W.1: Judgment/Noticing Task
- **Purpose:** Compare two equal groups visuals with different factor orders; trigger noticing that the NUMBER of groups differs while items per group also differs.
- **Visual: Context Visualizations (Mode 2).** Side-by-side arrangement. Concrete context (bags with items). \[NEW\] initialization. Multiple Choice interaction.
- **Guide:** "Look closely at both. What's DIFFERENT about these two?"
- **Prompt:** "What's different about these groups?"
- **Student Action:** Multiple Choice selection.
	- **Options:** A: Different number of bags ✓, B: Different items inside, C: One has more total items inside, D: Nothing—they're the same
- **Correct Answer:** A (Different number of bags)
- **On Correct:** "Right. Different number of bags. The left has 3 bags. The right has 5 bags."
- **Remediation:** Pipeline
> **Design Note:** This is a judgment task (comparison requiring thinking) rather than pure recognition. Both visuals show equal groups, so student must identify the specific difference.
---
### Interaction W.2: Personalization + Recognition
- **Purpose:** Activate M7 skill (matching expressions to visuals) and personalize with callback language.
- **Visual: Context Visualizations (Mode 2).** Single visualization. Concrete context (boxes with items). \[NEW\] initialization. Multiple Choice interaction. Method A.
- **Guide:** "Last time, you matched expressions to equal groups like these. Let's see if you remember how."
- **Prompt:** "Which expression matches these boxes?"
- **Student Action:** Multiple Choice selection via Equation Builder Method A.
	- **Options:** A: 4 × 2 ✓, B: 2 × 4 (reversal), C: 4 + 2 (addition), D: 2 + 4 (addition reversal)
- **Correct Answer:** A (4 × 2)
- **On Correct:** "You matched it: 4 times 2."
- **Remediation:** Pipeline
> **Design Note:** Changed visual (\[NEW\] state) to maintain engagement and avoid repetition. Uses different values (4×2) than opening hook (3×5 and 5×3).
---
**Bridge to Lesson**
- **Visual:** Expression "4 × 2" remains visible from W.2
- **Guide:** "Last time, you picked the right expression from choices. This time? YOU build them. Let's go."
- **No student action**
> **Design Note:** Session-relative language ("last time/this time"), creates anticipation without teaching, previews what Lesson will deliver (building methods).
---
### Warmup Verification Checklist
**Structure:**
- [ ] Hook in first 15-20 seconds (two visuals side-by-side, mystery setup)
- [ ] 2 meaningful interactions requiring student response (W.1 judgment, W.2 recognition)
- [ ] 1+ judgment/noticing task (W.1 — comparing two visuals with different factor orders)
- [ ] Bridge creates anticipation without teaching (promises "YOU get to build")
- [ ] Total time 3-5 minutes
**Engagement Anchors:**
- [ ] Narrative Setup (W.1 hook — "something interesting is hiding here")
- [ ] Choice/Agency (W.1 judgment task — student decides what's different)
- [ ] Personalization (W.2 — "Last time, you matched..." callback to M7)
**Constraints:**
- [ ] "Groups" / "groups of" / "times" used naturally (M7 vocabulary)
- [ ] NO "expression" as formal term (Lesson 1.1 introduces formally)
- [ ] NO "build the expression" prompts (Lesson teaches building)
- [ ] NO Methods C/D tiles (Lesson introduces these)
- [ ] NO circles context (Lesson transfer phase)
- [ ] Session-relative language only ("last time/this time")
- [ ] Light remediation only
- [ ] All tasks use M7 skills (recognition, matching)
- [ ] Cognitive load 20-30%
