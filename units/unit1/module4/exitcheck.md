## **1.8 EXIT CHECK (3-4 minutes)**

**Detail Level:** Full interaction specification (AI formats and tags)

### **Purpose**

Verify Lesson understanding before Practice. Tests whether student can read bar graph values at axis lines AND between lines (interpolation), create bars with scale of 10, and solve comparison problems using scaled bar graphs.

### **Parameters**

| Element | Specification |
| :---- | :---- |
| Problems | 4 |
| Cognitive Types | CREATE, IDENTIFY, and COMPARE (M4 can use all three) |
| Time | 3-4 minutes |
| Tone | Calm, low-stakes, neutral |
| Remediation | Full L-M-H added via Remediation Pipeline |

### **Constraints**

| MUST | MUST NOT |
| :---- | :---- |
| Use bar graphs only (primary M4 format) | Use picture graphs (observation only in M4) |
| Use vertical orientation (reduce variables) | Switch to horizontal during Exit Check |
| Use scale of 10 only | Mix scales |
| Include values BETWEEN lines (multiples of 5) | Skip interpolation assessment OR interpolate to values other than multiples of 5 |
| Use Data Table as data source for EC.3 | Introduce new data formats |
| Use click-to-set interaction (matches Lesson) | Add new interaction types |
| Match Lesson difficulty (values 20-60) | Use values outside Lesson range |

**Orientation Note:** Exit Check uses vertical orientation to minimize cognitive load during assessment. Although Lesson Section 3 taught horizontal, Exit Check reverts to vertical.

### **Alignment Check**

| Problem | Cognitive Type | Tests | Lesson Source |
| :---- | :---- | :---- | :---- |
| EC.1 | IDENTIFY | Read value AT axis line (multiple of 10) | Section 1: Interactions 1.2-1.3 |
| EC.2 | IDENTIFY | Read value BETWEEN lines (interpolation) | Section 2: Interactions 2.1-2.2 |
| EC.3 | CREATE | Create bars including between-lines values | Sections 1-2: Interactions 1.4-1.5, 2.3-2.5 |
| EC.4 | COMPARE | Solve "how many more" comparison problem | Section 3: Interactions 3.4-3.5 |

---

### **Transition into Exit Check**

- **Visual:** Lesson graph clears. Clean transition.
- **Guide:** "You made bar graphs with scale of 10. You read values at the lines AND between the lines. Let's see what you know."

**Voice Note:** Low-stakes framing. Brief, calm. No "test" language.

---

**Problem EC.1: Read Value at Axis Line (IDENTIFY)**

- **Visual: Bar Graphs (Mode 1: Reading).** Vertical. "Books Checked Out" — Nonfiction=30, Fiction=40, Comics=50, Poetry=20. 4 categories. Scale of 10, axis 0–60. All values at axis lines.
- **Guide:** "Look at the Fiction bar. What value does the Fiction bar show?"
- **Prompt:** "What value does the Fiction bar show?"
- **Student Action:** \[Multiple choice: 4, 35, 40, 45\]
- **Correct Answer:** 40
- **Answer Rationale:**
  - 40 = Correct (bar ends at 40 line)
  - 4 = Counted something incorrectly
  - 35 = Between-lines error (not applicable—bar IS at line)
  - 30 = Misread axis / Wrong bar
- **On Correct:** "40. The bar ends right at the 40 line on the axis."
- **On Incorrect:** \[Remediation Pipeline\]

**Alignment:** Tests same skill as Lesson 1.2-1.3 (reading bar height at axis line).

---

**Problem EC.2: Read Value Between Lines — Interpolation (IDENTIFY)**

- **Visual: Bar Graphs (Mode 1: Reading).** Vertical. "Points Earned" — Team A=35, Team B=50, Team C=25, Team D=40. 4 categories. Scale of 10, axis 0–60. Ghost gridlines at 5s visible.
- **Guide:** "Look at Team A's bar. It ends between the lines. The bar is between 30 and 40. What value does this show?"
- **Prompt:** "What is the value represented for Team A?"
- **Student Action:** \[Multiple choice: 30, 34, 35, 40\]
- **Correct Answer:** 35
- **Answer Rationale:**
  - 35 = Correct (halfway between 30 and 40)
  - 30 = Read lower line only (Misconception #5)
  - 34 = Random guess
  - 40 = Read upper line only
- **On Correct:** "35. It's halfway between 30 and 40. Half of 10 is 5."
- **On Incorrect:** \[Remediation Pipeline\]

**Alignment:** Tests same skill as Lesson 2.1-2.2 (reading interpolated value using "half of 10 = 5").

---

**Problem EC.3: Create Bar Graph with Mixed Values (CREATE)**

- **Visual: Data Tables (Numeric Display).** "Art Supplies Collected" — Markers=30, Crayons=45, Paintbrushes=20, Glue Sticks=55. Horizontal data table.
- **Visual: Bar Graphs (Mode 2: Creating).** Vertical. Scale of 10, axis 0–60. NO bars pre-filled. Ghost gridlines at 5s visible. Snap-to-5 enabled. Click-to-set.
- **Guide:** "Create the bar graph. Some values are at the lines, some are between."
- **Prompt:** "Complete the graph."
- **Student Action:** Set Markers=30 (at line), Crayons=45 (between 40-50), Paintbrushes=20 (at line), Glue Sticks=55 (between 50-60)
- **Correct Answer:** All four bars correctly placed
- **Answer Rationale:**
  - Markers at 30, Paintbrushes at 20 = at axis lines
  - Crayons at 45 (between 40-50), Glue Sticks at 55 (between 50-60) = between lines
- **On Correct:** "Markers and Paintbrushes at the lines. Crayons and Glue Sticks between the lines. You placed all four correctly."
- **On Incorrect:** \[Remediation Pipeline\]

**Alignment:** Tests same skill as Lesson 1.4-1.5 and 2.3-2.5 (creating bars at lines AND between lines).

---

**Problem EC.4: Comparison Problem (COMPARE)**

- **Visual: Bar Graphs (Mode 2: Creating).** Completed graph from EC.3. Markers=30, Crayons=45, Paintbrushes=20, Glue Sticks=55. Data Table not visible.
- **Guide:** "Use the graph you made. How many MORE art supplies does Glue Sticks have than Markers?"
- **Prompt:** "How many MORE art supplies does Glue Sticks have than Markers?"
- **Student Action:** \[Multiple choice: 15, 25, 55, 85\]
- **Correct Answer:** 25
- **Answer Rationale:**
  - 25 = Correct (55 − 30 = 25)
  - 15 = Miscalculation
  - 55 = Glue Sticks value only
  - 85 = Added instead of subtracted
- **On Correct:** "25 more. Glue Sticks has 55, Markers has 30. The difference is 25."
- **On Incorrect:** \[Remediation Pipeline\]

**Alignment:** Tests same skill as Lesson 3.4-3.5 (comparison using scaled bar graph).

---

### **Exit Check Closure**

- **Visual:** Completed graph remains briefly, then clears.
- **Guide:** "You can read bar graphs with scale of 10—even values between the lines. You're ready to practice."

**Voice Note:** Brief, warm. No overpraise. Transition to Practice.
