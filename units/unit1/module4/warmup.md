## **1.6 WARMUP (3-4 minutes)**

**Detail Level:** Structure + parameters (AI writes dialogue)

---

### **Purpose**

Activate prior knowledge of scaled bar graphs (1:5) from M3 while introducing scale of 10 as "even more efficient." Students collect personal data with scale-10 values, see their data as BOTH a 1:10 picture graph (their only exposure to this format) AND a 1:10 bar graph. The discovery that "scale of 10 uses even fewer symbols" creates motivation for Lesson's focus on bar graph creation with the new scale.

### **Parameters**

| Element | Specification |
| :---- | :---- |
| Interactions | 4 interactions (Game + Scale Comparison + Picture-to-Bar Bridge + Lesson Bridge) |
| Active Participation | 3 of 4 require student response |
| Format | Data Collection Game → Scale Comparison (1:5 vs 1:10) → Picture/Bar Comparison → Bridge |
| Cognitive Load | 20-30% (observation and comparison are accessible) |
| Remediation | Light only |
| Tone | Medium-High energy, curious, builds anticipation |
| Engagement Anchors | Narrative Setup (Minis), Personalization (their data), Curiosity Gap (which is more efficient?) |

### **Constraints**

| USE | AVOID |
| :---- | :---- |
| Informal language ("Check this out," "Watch what happens") | Formal phrasing ("Observe the transformation") |
| "Scale of 5" and "Scale of 10" (familiar pattern from M2-M3) | "Axis" vocabulary (save for Lesson) |
| "Count by 10s" casually | Formal skip-counting instruction |
| Observable acknowledgments ("You counted 40") | Assumed states ("You noticed the pattern") |
| 3-4 categories | 2 or 5+ categories |
| Multiples of 10 only (20, 30, 40, 50, 60) | Non-multiples of 10 (interpolation is Lesson content) |
| Picture graph shown for comparison | Student creating picture graph |
| Bar graph shown alongside | Formal bar graph creation |

---

**Interaction W.1: Data Collection Game**

- **Purpose:** Generate personalized data with scale-10 values. Create investment through Minis context.
- **Hook (first 15-20 seconds):**
  - **Visual:** Minis characters appear in animated counting scenario.
  - **Guide:** "The Minis are collecting items! Help count what they found."
- **Task:**
  - Student counts Minis/items across 3-4 categories
  - Results generated in multiples of 10 (20, 30, 40, 50, 60 range)
  - Values should show clear differences for later comparison
- **On Complete:**
  - **Guide:** "You counted \[results\]. That's a lot! Let's see how to show this data."
- **Remediation (Light):** If count is off, Guide reveals correct count: "Let's check—that's \[X\]."
- **Duration:** ~60 seconds

---

**Interaction W.2: Scale Comparison (1:5 vs 1:10)**

- **Purpose:** Activate M3 knowledge while showing scale of 10 is even more efficient. Creates "aha" moment about efficiency progression.
- **Visual:** Picture Graphs (side-by-side display):
  - LEFT: Student's data as 1:5 vertical picture graph (each symbol = 5)
  - RIGHT: Same data as 1:10 vertical picture graph (each symbol = 10)
  - Keys visible on both: "Each ⭐ = 5" and "Each ⭐ = 10"
- **Guide:** "Here's your data two ways. Look at both graphs. Which graph uses FEWER symbols to show the same data?"
- **Prompt:** "Which graph uses FEWER symbols to show the same data?"
- **Student Action:** Select one of the two graphs
- **Correct Answer:** The 1:10 graph (right)
- **On Correct:** "Scale of 10 uses fewer symbols! Each symbol shows more, so you need fewer of them."
- **On Incorrect:** "Look at the keys. Scale of 5 means each symbol shows 5. Scale of 10 means each symbol shows 10. Which needs fewer symbols for the same amount?"
- **Guide (efficiency reinforcement):** "With scale of 10, each symbol shows more than scale of 5. That's why you need fewer symbols for the same data!"
- **Duration:** 45-60 seconds

---

**Interaction W.3: Picture-to-Bar Bridge (1:10)**

- **Purpose:** Show that bar graphs display the same data as picture graphs. This is student's ONLY exposure to 1:10 picture graphs—they won't create them.
- **Visual:** Side-by-side display:
  - LEFT: 1:10 picture graph (from W.2)
  - RIGHT: 1:10 bar graph showing SAME data
  - Both vertical orientation
  - Bar graph shows axis with scale of 10 (0, 10, 20, 30, 40, 50, 60)
- **Guide:** "Now watch this. Here's the SAME data as a bar graph."
- **Guide:** "This category has \[X\]. The picture graph shows \[X÷10\] symbols. The bar graph shows a bar up to \[X\]. Both graphs show \[Y\] for \[second category\]. Can you find where?"
- **Prompt:** "Both graphs show \[Y\] for \[second category\]. Can you find where?"
- **Student Action:** Click on the bar OR the symbols for that category
- **Correct Answer:** Either correct location accepted
- **On Correct:** "Same data! Picture graphs use symbols. Bar graphs use bars. Both show \[Y\]."
- **Guide:** "Bar graphs are really useful with scale of 10. You read the HEIGHT instead of counting symbols."
- **Duration:** 45-60 seconds

**Note:** This is student's ONLY interaction with 1:10 picture graphs. They observe but do not create.

---

**Interaction W.4: Bridge to Lesson**

- **Purpose:** Create anticipation for Lesson without teaching. Frame the shift to bar graph CREATION.
- **Visual:** Bar graph from W.3 remains visible. Picture graph removed.
- **Guide:** "You've seen scale of 10 with picture graphs and bar graphs. Both show the same data."
- **Guide:** "Bar graphs are powerful—you can read exact amounts from the numbers on the side."
- **Bridge Statement:** "Ready to learn how to CREATE bar graphs with scale of 10? Let's go!"
- **No student prompt** (transition to Lesson)
- **Duration:** 15-20 seconds

---

### **Verification Checklist**

**Structure:**

- [ ] Hook appears in first 15-20 seconds (Minis appear)
- [ ] 2+ engagement anchors (Narrative, Personalization, Curiosity Gap)
- [ ] 3 active student interactions (Game, Scale Comparison, Picture-Bar Bridge)
- [ ] Clear bridge to Lesson creates anticipation
- [ ] Total time 3-4 minutes

**Content Alignment:**

- [ ] 3-4 categories (not 2 or 5+)
- [ ] All values multiples of 10 (20, 30, 40, 50, 60 range)
- [ ] No non-multiples (interpolation is Lesson content)
- [ ] Scale of 5 vs. Scale of 10 comparison shown
- [ ] Student sees 1:10 picture graph (their only exposure)
- [ ] Student sees 1:10 bar graph alongside picture graph
- [ ] "Scale of 10 uses fewer symbols" discovery is explicit
- [ ] Bar graph framed as primary format for Lesson
- [ ] Keys visible on both graph types

**Constraints:**

- [ ] No student graph creation (observation and comparison only)
- [ ] No "axis" vocabulary (save for Lesson)
- [ ] No interpolation / "between the lines" content
- [ ] No formal skip-counting instruction
- [ ] Cognitive load light (20-30%)
