## **1.9 SYNTHESIS (5-6 minutes)**

**Detail Level:** Structure \+ parameters (AI writes dialogue)

---

### **Purpose**

Consolidate understanding that picture graphs and bar graphs are two ways to show the same data. Connect graph reading to real-world contexts. Reflect on problem-solving strategies. This is NOT more practice—it's connection-making and consolidation.

---

### **Parameters**

| Element | Specification |
| :---- | :---- |
| Sections | 4 (3 connection tasks \+ 1 metacognitive reflection) |
| Task Types | Type A (Pattern Discovery), Type D (Representation Transfer), Type C (Real-World Bridge) |
| Remediation | Light only—mastery assumed |
| Tone | Reflective, warm, discovering together |
| Energy | Medium (not exhausted, not high-energy) |

---

### **Constraints**

| DO | DO NOT |
| :---- | :---- |
| Connection tasks (pattern, transfer, real-world) | More practice problems |
| Name principles students discovered | Introduce new procedures |
| Light remediation if needed | Extended teaching or heavy remediation |
| Specific identity closure | Generic praise ("Great job\!") |
| Visual support for every task | Abstract discussion without visuals |
| Use picture graphs and bar graphs from Lesson | Introduce new graph types or scaled graphs |

---

### **Opening Frame**

* **Purpose:** Signal shift from Exit Check to reflection/connection.
* **Guide:** "You read picture graphs and bar graphs and used them to answer questions about the data. Now let's connect some ideas."

---

**Task S.1: Representation Transfer (Type D)**

* **Purpose:** Surface understanding that picture graphs and bar graphs show the same information differently.
* **Visual: Picture Graphs (Mode 1: Reading) \+ Bar Graphs (Mode 1: Reading).** Side-by-side display. Same data: "Favorite Sports," 4 categories, 1:1 scale. Picture graph on left, bar graph on right. Both have visible keys/axis labels. Data Table not visible.
* **Guide:** "Look at these two graphs. They show the SAME data. What's the same about both graphs? Select all that apply."
* **Prompt:** "What's the same about both graphs? Select all that apply."
* **Student Action:** \[Multi-select\]
  * ☐ They have the same title
  * ☐ They show the same categories
  * ☐ Soccer has the most in both
  * ☐ They look exactly the same
* **Correct Answers:** First three selected (not fourth)
* **On Correct:** "The information is the same. The format is different."
* **Connection (Guide states):** "Picture graphs use symbols. Bar graphs use bars. Both show the same data—you can read either one."

**Alignment:** Connects Section 1 (picture graphs) and Section 2 (bar graphs) from Lesson.

---

**Task S.2: Pattern Discovery (Type A)**

* **Purpose:** Consolidate understanding that each graph component has a specific job.
* **Visual: Picture Graphs (Mode 1: Reading).** "Favorite Pets" graph with 3 components labeled by system overlay: A \= Title ("Favorite Pets"), B \= Key ("Each ★ \= 1 vote"), C \= Category labels (Dogs, Cats, Fish, Birds). Labels are display-only; student clicks to select A, B, or C. Data Table not visible.
* **Guide:** "Every part of a graph has a job. This picture graph has three parts labeled A, B, and C. Click on the part that shows what each symbol means."
* **Prompt:** "Which part tells you what each symbol means?"
* **Student Action:** Click to select (A, B, or C)
* **Correct Answer:** Component B (the Key)
* **On Correct:** "The key tells you what each symbol means. Here, each star equals 1 vote."
* **Connection (Guide states):** "Title tells what the graph is about. Labels name the categories. Key shows the scale. Every part has a job."

**Alignment:** Reinforces component vocabulary from Lesson Section 1 (Section 1.3).

---

**Task S.3: Real-World Bridge (Type C)**

* **Purpose:** Connect graph reading to contexts outside math class.
* **Visual:** Three real-world scenarios displayed (no graph toy). Scenario 1: "Your class votes on a field trip destination." Scenario 2: "Library tracks which book genre students check out most." Scenario 3: "Cafeteria tracks which lunch is most popular." Multi-select enabled. Data Table not visible. No graph toys active.
* **Guide:** "Graphs aren't just in math class. People use them everywhere. Look at these three scenarios. Which of these could use a bar graph to show the data? Select all that apply."
* **Prompt:** "Which of these could use a bar graph to show the data? Select all that apply.
* **Student Action:** \[Multi-select\]
  * ☐ Scenario 1 (field trip votes)
  * ☐ Scenario 2 (library book genres)
  * ☐ Scenario 3 (lunch popularity)
* **Correct Answers:** All three (all are categorical data that could use bar graphs)
* **On Correct:** "All of them\! Any time you're counting and comparing categories, a graph can help."
* **Connection (Guide states):** "Graphs help people see patterns and make decisions. You'll see them in school, in stores, on the news—everywhere."

**Alignment:** Extends Lesson understanding to transfer contexts.

---

#### **Metacognitive Reflection: Comparison Questions**

**Purpose:** Surface the pattern that BOTH "how many more" and "how many fewer" require subtraction.

* **Visual:** Two comparison questions displayed as text (no graph toy active): "How many MORE students chose dogs than cats?" and "How many FEWER students chose fish than birds?" Data Table not visible.
* **Guide:** "You saw two kinds of comparison questions today. To answer both questions, what do you need to do?"
* **Prompt:** "To answer both questions, what do you need to do?"
* **Student Action:** \[Multiple choice\]
* Add the categories
* Subtract the categories
* Find each value
* **Correct Answer:** Subtract the categories
* **On Correct:** Guide: "'How many more' AND 'how many fewer' — both mean subtract. You find the difference to compare."
* **On Incorrect (Add the categories):** Guide: "'In all' means add. But 'more' and 'fewer' are comparing. What operation finds the difference?"
* **On Incorrect (Find each value):** Guide: "You do need to find each value first. But then what? How do you compare them?"

**Reflection Note:** This surfaces the key insight from Section 3 — both comparison question types use subtraction, despite different wording.

---

### **Identity-Building Closure**

* **Purpose:** Specific, growth-oriented affirmation connecting to future learning.
* **Visual: Picture Graphs (Mode 2: Creating) \+ Picture Graphs (Mode 1: Reading).** Student's Warmup graph (horizontal, 3 categories) reappears on left. A Lesson graph (from Section 1 or 2\) appears on right. Brief display — visual bookend connecting start to finish. Data Table not visible.
* **Guide:** "You made your own graph. You read other people's graphs. You answered comparison questions and found totals. You know how to collect and represent data."
* **Bridge to M2:** "Next time, you'll see graphs where each symbol means MORE than one. Get ready for a new challenge."
