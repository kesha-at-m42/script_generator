## **1.8 EXIT CHECK (3-4 minutes)**

**Detail Level:** Full interaction specification (AI formats and tags)

---

### **Purpose**

Verify Lesson understanding before Practice. Tests whether student can read graph values, identify most/least, solve "how many more/fewer" problems, and solve "in all" problems using graphs taught in Lesson.

---

### **Parameters**

| Element | Specification |
| :---- | :---- |
| Sections | 4 |
| Cognitive Types | IDENTIFY and COMPARE only (per Playbook: Modules 1-3 focus) |
| Time | 3-4 minutes |
| Tone | Calm, low-stakes, neutral |
| Remediation | Full L-M-H added via Remediation Pipeline |

---

### **Constraints**

| MUST | MUST NOT |
| :---- | :---- |
| Use picture graphs AND bar graphs (both taught) | Introduce new graph types |
| Use both orientations (both taught) | Use only one orientation |
| Use 1:1 scale only | Use scaled graphs (M2+) |
| Use values in similar range to Lesson (3-10) | Use larger/more complex values |
| Use same interaction type (multiple choice) | Add new interaction types |
| Match Lesson difficulty | Increase complexity beyond Lesson |

---

### **Alignment Check**

| Section | Cognitive Type | Tests | Lesson Source |
| :---- | :---- | :---- | :---- |
| EC.1 | IDENTIFY | Read value from picture graph | Section 1: Reading Picture Graphs (1.4) |
| EC.2 | IDENTIFY | Identify least/fewest | Section 2: Reading Bar Graphs (2.4) |
| EC.3 | COMPARE | Solve "how many more" (subtract) | Section 3: Section Solving (3.1, 3.2) |
| EC.4 | APPLY | Solve "in all" (add) | Section 3: Section Solving (3.3) |

---

### **Transition into Exit Check**

- **Visual:** Lesson graph clears. Clean transition.
- **Guide:** "You read picture graphs and bar graphs. You solved comparison problems and found totals. Let's see what you know."

**Voice Note:** Low-stakes framing. No "test" or "check if you can" language. Brief, calm.

---

**Section EC.1: Read a Value (IDENTIFY)**

- **Visual: Picture Graphs (Mode 1: Reading).** Vertical orientation. Context: "Popular Snacks," 4 categories. Key: "Each symbol \= 1 vote." Values: Popcorn=5, Pretzels=3, Crackers=7, Fruit=4. Data Table not visible.
- **Guide:** "Look at this Popular Snacks picture graph. How many students chose crackers?"
- **Prompt:** "How many students chose crackers?"
- **Student Action:** \[Multiple choice: 3, 4, 5, 7\]
- **Correct Answer:** 7
- **Answer Rationale:**
  - 7 \= Correct (Crackers value)
  - 3 \= Misread (Pretzels value)
  - 4 \= Misread (Fruit value)
  - 5 \= Misread (Popcorn value)
- **On Correct:** "7 students chose crackers."
- **On Incorrect:** \[Remediation Pipeline\]

**Alignment:** Tests same skill as Lesson 1.4 (read value from picture graph, vertical orientation).

---

**Section EC.2: Find the Fewest (COMPARE)**

- **Visual: Bar Graphs (Mode 1: Reading).** Horizontal orientation."Favorite Seasons," 4 categories. Scale \= 1\. Values: Spring=6, Summer=9, Fall=4, Winter=5. Axis marked 0-10 by 1s.
- **Guide:** "Look at this Favorite Seasons bar graph. Click on which season the FEWEST students like."
- **Prompt:** "Click on which season the FEWEST students like."
- **Student Action:** Click to select category
- **Correct Answer:** Fall (4)
- **On Correct:** "Fall got the fewest, only 4 votes."
- **On Incorrect:** \[Remediation Pipeline\]

**Alignment:** Tests same skill as Lesson 2.4 (identify fewest in bar graph, horizontal orientation).

---

**Section EC.3: How Many More (COMPARE)**

- **Visual: Bar Graphs (Mode 1: Reading).** Vertical orientation."Books Read This Week," 4 categories. Scale \= 1\. Axis marked 0-8 by 1s.Values: Ava=8, Ben=5, Cara=6, Dan=3. **Guide:** "How many more books did Ava read than Dan?"
- **Student Action:** \[Multiple choice: 3, 5, 8, 11\]
- **Correct Answer:** 5
- **Answer Rationale:**
  - 5 \= Correct (8 \- 3 \= 5, the difference)
  - 3 \= Error: Used Dan's value only
  - 8 \= Error: Used Ava's value only
  - 11 \= Error: Added instead of subtracted (8 \+ 3\)
- **On Correct:** "5 more books. Ava read 8, Dan read 3\. The difference is 5."
- **On Incorrect:** \[Remediation Pipeline\] \- Ava and Dan bars highlighted.

**Alignment:** Tests same skill as Lesson 3.1-3.2 ("how many more" \= subtract to find difference).

---

**Section EC.4: How Many In All (IDENTIFY)**

- **Visual: Bar Graphs (Mode 1: Reading).** Same "Books Read This Week" graph. Data Table not visible.
- **Guide:** "Look at this Books Read This Week bar graph. How many books did Ben and Cara read in all?"
- **Prompt:** ""How many books did Ben and Cara read in all?"
- **Student Action:** \[Multiple choice: 6, 9, 11, 14\]
- **Correct Answer:** 11
- **Answer Rationale:**
  - 11 \= Correct (5 \+ 6 \= 11, the total)
  - 6 \= Error: Used Cara's value only
  - 9 \= Error: Subtraction or misread
  - 14 \= Error: Included additional category
- **On Correct:** "11 books in all. Ben read 5, Cara read 6\. You added to find the total."
- **On Incorrect:** \[Remediation Pipeline\] \- Ben (5) and Cara (6) bars highlighted.

**Alignment:** Tests same skill as Lesson 3.3 ("in all" \= add to find total).

---

### **Exit Check Closure**

- **Visual:** Final graph remains or clears to transition screen.
- **Guide:** "You're ready to practice."

### **Verification Checklist**

**Structure:**

- [ ] Exactly 4 problems
- [ ] Transition references Lesson content with low-stakes framing
- [ ] Closure statement present (brief, no overpraise)
- [ ] Total time 3-4 minutes

**Alignment:**

- [ ] EC.1 tests reading values (Lesson Section 1\)
- [ ] EC.2 tests most/least comparison (Lesson Section 2\)
- [ ] EC.3 tests "how many more" subtraction (Lesson Section 3\)
- [ ] EC.4 tests "in all" addition (Lesson Section 3\)

**Constraints:**

- [ ] Uses picture graph AND bar graph from Lesson
- [ ] Uses both orientations (vertical and horizontal)
- [ ] Uses only 1:1 scale (no scaled graphs)
- [ ] IDENTIFY and COMPARE only (no CREATE per Modules 1-3 guidance)
- [ ] Difficulty matches Lesson (same value ranges, same question types)
- [ ] No new visual models or graph types introduced

**Voice Quality:**

- [ ] Feedback is brief and specific (5-10 words)
- [ ] No overpraise ("Perfect\!" "Amazing\!" "Excellent\!")
- [ ] No assumed internal states ("You thought carefully")
- [ ] Observable results only ("7 students chose crackers")
- [ ] Low-stakes tone throughout (no "test" language)
