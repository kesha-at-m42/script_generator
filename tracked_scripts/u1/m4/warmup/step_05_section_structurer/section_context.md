## s2_1_scale_comparison_1_5_vs
# Section Summary: s2_1_scale_comparison_1_5_vs

**VISUAL STATE:** Two vertical picture graphs displayed side-by-side in reading mode. Left graph (picture_graph_scale_5): scale of 5, categories Red Items, Blue Items, Yellow Items, Green Items with key "Each ⭐ = 5." Right graph (picture_graph_scale_10): scale of 10, same four categories, with key "Each ⭐ = 10." Both represent identical data (Red Items=20, Blue Items=30, Yellow Items=40, Green Items=50). Right graph is highlighted at section end.

**CONTENT:** Students learned how picture graph scale affects symbol quantity—a larger scale value (10 vs. 5) requires fewer symbols to represent the same data because each symbol represents a larger quantity. The concept that "each symbol shows more" with a higher scale was formally introduced.

**STUDENT ACTION:** Student clicked on the picture graph with scale of 10 to answer which graph uses fewer symbols to show the same data, demonstrating understanding that a scale of 10 is more efficient than a scale of 5 for this dataset.

## s3_1_picture_bar_bridge_1_10
# Section Summary: s3_1_picture_bar_bridge_1_10

**VISUAL STATE:** Two side-by-side graphs in reading mode display identical data with scale 10. Left: vertical picture graph with categories Red Items (2 symbols), Blue Items (3 symbols), Yellow Items (4 symbols), Green Items (5 symbols), each symbol = 10. Right: vertical bar graph with same categories, axis range 0–10–20–30–40–50–60, bars reaching Red Items=20, Blue Items=30, Yellow Items=40, Green Items=50. Yellow Items category is highlighted on both graphs at section end.

**CONTENT:** Students learned that picture graphs and bar graphs represent the same data in different formats—picture graphs use symbols (with a key indicating value per symbol), while bar graphs use bar height. The concept of scale 10 was reinforced: in picture graphs, each symbol represents 10; in bar graphs, the axis increments by 10, allowing students to read height directly rather than count symbols.

**STUDENT ACTION:** Student clicked to identify the Yellow Items category (value 40) on either graph, correctly locating where both representations show the same data point. The system highlighted Yellow Items on both graphs upon correct response.

## s4_1_bridge_lesson
# Section Summary: s4_1_bridge_lesson

**VISUAL STATE:** A vertical bar graph titled "Mini Items" is displayed in reading mode throughout the section. The graph contains four categories—Red Items, Blue Items, Yellow Items, and Green Items—with values of 20, 30, 40, and 50 respectively. The y-axis ranges from 0 to 60 in increments of 10 (scale = 10).

**CONTENT:** This transition section bridges picture graphs and bar graphs by establishing that both representations display identical data using the same scale of 10. The lesson introduces the key advantage of bar graphs: students can read exact numerical values directly from the axis labels, rather than counting symbols. The section concludes by previewing the next learning objective: creating bar graphs with a scale of 10.

**STUDENT ACTION:** The student passively observed the bar graph and listened to explanatory dialogue; no interactive construction or response was required in this transition beat.
