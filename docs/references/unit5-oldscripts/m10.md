### **LEARNING GOAL (VERBATIM):**

"Compare fractions with the same denominator using \>, \<, \= and justify with visual models."

### **CORE CONCEPTS:**

Same denominator comparison, comparison symbols (\>, \<, \=)

### **STANDARDS:**

**Building On:** 3.NF.A.3.C
 **Addressing:** 3.NF.A.3.D
 **Building Toward:** 3.NF.A.3.D (extended)

Greater than, less than, equal to (after visual grounding)

**\#8 \- Comparing different denominators:** Student tries to compare fractions without same denominators
 **\#10 \- Not recognizing comparison method:** Student doesn't recognize same denominator allows direct comparison
