# **MODULE 4 PATH B: DENOMINATORS AS INTERVAL MAKERS**

## **LEARNING GOAL (S):**

"Denominator \= number of equal intervals in the unit segment on the number line. Introduce non-unit fractions by iterating 1/b beyond 1."

## **CORE CONCEPTS:**

Denominator defines equal partitions; fractions as positions on number line; extending beyond 1

## **STANDARDS:**

**OUR Lesson 5 coverage:**

* Building On: 3.G.A.2, 3.NF.A.1
* Addressing: 3.NF.A.2a
* Building Toward: 3.NF.A.2b
  **OUR Lesson 6 coverage:**
* Building On: 3.NF.A.2a, 3.NF.A.2b
* Addressing: 3.NF.A.2b
* Building Toward: 3.NF.A.3a
  **Combined:** 3.G.A.2, 3.NF.A.1, 3.NF.A.2a, 3.NF.A.2b

## **VOCABULARY STAGING:**

Connect denominator (already learned in Module 3\) to equal intervals on the number line. Reinforce that numerator counts these intervals from zero.

## **MISCONCEPTIONS TO ADDRESS:**

\#2: Misidentifying the Whole \- Maintaining clarity about what represents "1" on the number line
\#4: Improper Spacing on Number Line \- Spacing units evenly and consistently
\#5: Counting Tick Marks Instead of Spaces \- Understanding intervals (spaces) not ticks define fractions

# **PRACTICE PHASE**

**Duration:** \~10-12 minutes
 **Learning Goal:** "Denominator \= number of equal intervals in the unit segment on the number line. Introduce non-unit fractions by iterating 1/b beyond 1."
 **Primary Visual Tool:** Number line

### **LEARNING GOAL COMPONENTS**

* **Component A:** Count and create equal intervals based on denominator \- CREATE (35% weight, min 75% to pass)
* **Component B:** Identify denominators and understand interval relationships \- IDENTIFY/COMPARE (30% weight, min 60% to pass)
* **Component C:** Apply interval understanding to place fractions and extend beyond 1 \- APPLY (35% weight, min 50% to pass)

### **COGNITIVE TYPE MAPPING FOR MODULE 4 (FOUNDATIONAL)**

Per Playbook for Modules 1-4:

* **CREATE:** Problems 1, 5 (Component A)
* **IDENTIFY:** Problems 2, 6, 7 (Component B)
* **COMPARE:** Problem 3, 9 (Component B)
* **APPLY:** Problems 4, 8 (Component C \- maximum 2 for foundational)
* **CONNECT:** Not included in foundational modules

### **CRITICAL GATES**

* Must create at least 1 set of equal intervals correctly
* Must identify denominator from intervals at least once
* Must pass at least 1 APPLY problem (placing fractions on number line)
* No persistent confusion between marks and spaces
