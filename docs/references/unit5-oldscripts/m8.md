# **MODULE 8 PATH B (SINGAPORE)**

## **MODULE INFORMATION**

### **LEARNING GOALS (VERBATIM):**

"Identify equivalent fractions; two fractions are equivalent if they are the same size and refer to the same whole." "Generate and justify simple equivalent fractions using visual models."

### **CORE CONCEPTS:**

"Visual equivalence through multiple representations"

### **STANDARDS:**

**Building On:** 3.NF.A.1 **Addressing:** 3.NF.A.3a, 3.NF.A.3b **Building Toward:** 3.NF.A.3d

### **VOCABULARY STAGING SEQUENCE:**

"Introduce 'equivalent,' 'same whole,' 'same size' after visual discovery" "Introduce 'generate' after students create equivalent fractions"

**SME Note:** Students don’t need to know the word justify just because it’s in the standard. It means they need to be able to explain it. ½ and 2/4 are the same because…

### **MISCONCEPTIONS TO ADDRESS:**

**\#7 \- Difficulty Recognizing Equivalence:** "Not seeing that 2/4 \= 1/2, even when visually aligned" **\#6 \- Reversing Numerator and Denominator:** "Swapping top/bottom when labeling or reading fractions" **\#10 \- Overgeneralizing Rules:** "Applying rules like 'larger number \= bigger piece' across contexts"
