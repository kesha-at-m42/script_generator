## §PT.2 Goal Decomposition

### Spine Skills Active in M5

| Skill ID | Status | Description | Component | Verb | Representations |
|----------|--------|-------------|-----------|------|-----------------|
| ReadBarGraph | PRC | Read values from bar graphs at various scales | conceptual | identify | Bar Graphs |
| SelectScale | INT | Choose appropriate scale (2, 5, or 10) for a dataset | transfer | apply | Bar Graphs |
| CompareData | PRC | Compare data as part of scale evaluation | conceptual | compare | Bar Graphs |
| CompareData:difference | PRC | "How many more/fewer" with various scales | conceptual | compare | Bar Graphs |

**Key M5 characteristic:** Meta-cognitive module. Students make STRATEGIC decisions, not just execute procedures. SelectScale is the only new skill; everything else is practiced.

**Skills present but NOT separately templated:** CompareData:ordinal and CompareData:combination appear in the TF as embedded questions ("Now tell me: which category has the most?" after scale selection). These are used as context for SelectScale, not as standalone targets. CompareData:difference gets dedicated coverage because the TF explicitly includes "How many more X than Y?" as independent questions.

### Open Questions / Flags

1. **Justification depth:** SME feedback says "over-emphasis on justifying" is a concern for 3rd graders. Simple justifications acceptable: "it looks better," "I can see the data." Templates should keep justification as MC, not free-response. Stretch/challenge can have more nuanced options.

2. **SelectScale is single-module (M5 only):** This means all tiers must be represented within one module's template set. No cross-module progression for this skill. Confidence tier should feel approachable for students who may struggle with the meta-cognitive demand.

3. **CompareData as embedded vs standalone:** The TF shows comparison questions embedded WITHIN scale selection activities (answer questions AFTER choosing scale). Some templates should integrate both (select scale → answer question) while others can isolate reading/comparison from scale choice.
