### Misconception Targeting

| Misconception | ID | Relevant Skills | Priority | Template Coverage |
|---------------|----|-----------------|---------|--------------------|
| Groups vs. Size Confusion | 10 | WriteMultExpression, InterpretMultExpression | PRIMARY | Factor reversal: writes 3 × 4 for 4 groups of 3. MC distractors with reversed factors. Core detection template for both directions. |

### Remediation Notes (from TF)

The TF includes an explicit 3-level remediation pattern for M8:
- **Light:** Guide highlights groups visually. "Count carefully. How many groups?"
- **Medium:** Numbers appear under each group (4 4 4). "Count the fours. There are 3 fours." No addition signs.
- **Heavy:** Full visual progression — groups highlighted one at a time, numbers appear, guide counts, expression builds.

This maps well to RDR Non-MC remediation (L-M-H). Templates should reference these as scaffold/remediation anchors.

### Open Questions / Flags

1. **MC first, then tiles, then all methods:** The method progression is a scaffolding arc within WriteMultExpression. Confidence/support should use MC (recognition before construction). Baseline uses tiles. Stretch/challenge uses all methods.

2. **Bidirectional mixed templates:** The TF late activities explicitly mix both directions. Stretch/challenge templates that ask "Look at this visual. Write the expression. Now: here's an expression — which visual matches?" would test bidirectional fluency in a single interaction.

3. **No addition signs until Synthesis:** The TF explicitly states "Addition signs (4 + 4 + 4) not introduced until M8 Synthesis, then applied in M9." Templates should NOT include addition expressions as correct answers or options. Only multiplication notation.
