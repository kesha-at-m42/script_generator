### Open Questions / Flags

1. **Observation phase (Activities 1-2):** TF has a non-creation observation phase. These activities are lesson-only — students watch transformations. No Practice templates needed for observation, but it means students have 2 fewer creation repetitions than if they jumped straight in. SME noted this is valuable given the double shift.

2. **InterpretHalfScale EXT vs INT:** Spine says EXT because concept was introduced in M2. But the representation is new (bar interpolation vs picture half-symbols). Templates should treat it as "new application of known concept" — confidence tier should reference the M2 connection explicitly ("Remember half-symbols? Same idea").

3. **SME feedback on gridlines:** Andrea says students need to adjust ALL bars with multiples of 10 before dealing with non-multiples. Activity 3-4 = all multiples of 10. Activity 5+ = introduce halfway. Tier mapping should respect this scaffolding arc.
