## Key Design Decisions (Unit-Level)

Resolved in the TVP. Do not re-litigate unless new evidence surfaces.

- **D1 — Unified path.** Single instructional path with adaptive scaffolding intensity. NOT multi-path differentiation.
- **D2 — Both division types from Day 1.** Partitive (sharing) and quotitive (grouping) taught with EQUAL emphasis from M1. Not partitive-first.
- **D3 — Array tool extends Unit 1.** Extend the existing Unit 1 arrays tool with division-specific features (circling groups, showing leftover). NOT a new tool.
- **D4 — Simultaneous CRA connections.** NOT strict sequential gates. Students connect concrete, representational, and abstract within the same interaction when possible.
- **D5 — Properties naming.** Commutative named informally ("order doesn't matter"). Distributive and associative taught as strategies only — NO formal property names at Grade 3.
- **D6 — NEVER "add a zero."** Place value reasoning always. 4 × 30 = 4 × 3 tens = 12 tens = 120. The phrase "add a zero" is forbidden in all Guide dialogue.
- **D7 — Unknown representation scaffold.** Progressive: `?` (M1–M3) → `□` (M3–M4) → letters (M10+). All three remain accepted once introduced. Cumulative flexibility.
- **D8 — Equals sign as presentation pattern.** True/false number sentences ELIMINATED as standalone activities. Nonstandard equation formats (e.g., `? = 3 × 4`) used naturally in presentation.
- **D10 — Multiplication fluency diagnostic gate before M11.** 80% accuracy threshold on basic facts (×1 through ×10). Students below threshold get remediation before larger-number division.
- **D13 — M4/M5 split.** Original M4 (L7–L9) was too packed. Split into M4 (fact families) + M5 (table patterns + commutativity).

### Special Considerations for This Unit

1. **Division is brand new.** Unlike Unit 3 (which built on Grade 2 add/sub), students have never formally divided. M1 must be heavily concrete — the TVP specifies CONC 70%.
2. **The inverse relationship is THE module.** M3 is the most important module in the unit. If students don't connect × and ÷ here, M4–M13 all suffer. Allocate extra review time.
3. **"Add a zero" will surface in M7.** Students are "nearly guaranteed" (SME) to encounter this from home. The Guide must address it directly with place value reasoning, not just avoid saying it.
4. **Diagnostic gate before M11.** Students need 80% multiplication fact fluency before attempting larger-number division. The remediation pipeline handles students below threshold.
5. **Tonal shift at M10.** Word problems feel very different from the strategy/fluency work in M5–M9. The bridge from M9 → M10 needs to prepare students for this shift.
6. **M11 re-activation challenge.** Students haven't done concrete division since M2–M3. M11 needs explicit re-activation, not assumption of retention.
7. **No non-template artifacts in the SP body.** Dimension tracking tables, balance calculations, data summaries, and authoring-only annotations belong in Working Notes, not the Starter Pack. M2 inserted inline Data Summary tables and Balance blocks into §1.7 that had to be removed during post-Gate 4 cleanup. The SP body should contain only template-defined sections and fields.
8. **Assessment vocabulary in §1.3 must be formally introduced by that module.** Do not list terms the student hasn't been formally exposed to yet, even if they appear on the state test. Map state-test vocabulary to the module that formalizes it, not the module that previews it. (From cross-unit review: M2 listed "quotient" as assessment vocab before M3 formalized it.)
