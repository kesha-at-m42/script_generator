# Visual Constraints - Module 1 Path C (VPSS)

## Tangible Types
- Pattern Tiles (triangles, squares, hexagons)
- Grid Arrays (rectangular grids)

## Description
Path C uses visual-first, spatial reasoning with pattern tiles and grid arrays. Students explore how shapes fit together, discover spatial relationships visually, and verify equality through visual patterns.

## Constraints
- Pattern tiles can be manipulated, arranged, and compared
- Grid arrays show equal parts through visual patterns (e.g., 2x3 grid, 2x4 grid)
- Tiles can fit together to form larger shapes (e.g., two triangles make a square)
- Visual overlays can be used to verify equality
- Maximum of 4 visual elements can be shown for comparison
- Grids can show 2, 3, 4, 6, or 8 equal sections
- Pattern tiles emphasize spatial relationships and visual composition
